# Python Weather Scraper

An automated web scraping tool designed to collect and log local weather data using Python and Playwright.

## 🚀 Features
* **Automated Data Collection:** Uses Playwright to navigate and extract real-time weather information.
* **Data Persistence:** Automatically saves and appends data to a structured `weather_history.csv` file.
* **Headless Execution:** Designed to run efficiently in the background.

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **Library:** Playwright (Web Automation)
* **Storage:** CSV (Data Logging)

## 📂 Project Structure
* `weather_scraper.py`: The main script for navigation and extraction.
* `weather_history.csv`: The output file containing logged weather trends.
